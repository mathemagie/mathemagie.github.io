Nabaztag ISS


My Nabaztag wiggles as the International Space Station flies over France!
Here's the python source code to create this enchanted object: https://raw.githubusercontent.com/mathemagie/nabaztag_iss/main/ear.py
The code repeatedly checks the ISS's location and, if it's over France, sends ear-movement commands to a Nabaztag robot via a local TCP connection.

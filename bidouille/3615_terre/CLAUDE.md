# CLAUDE.md - 3615 TERRE Agent Memory

## UI Language

All user-facing application text must be written in English.

- Do not add French copy to visible UI, status messages, overlays, buttons, labels, aria labels, or tooltips.
- If a neutral status message is needed, use English wording such as `RECEIVING SIGNAL...`.
- Keep the location-spoiler rule from `AGENTS.md`: do not reveal city, country, region, coordinates, or distance by default.
